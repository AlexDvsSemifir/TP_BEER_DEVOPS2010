### CODE 1

```sql
--- USE beer

SELECT annee, numero_ticket, date_vente
FROM ticket
WHERE DATE_FORMAT(date_vente, "%Y-%m") = "2014-03"

```

### CODE 2

```sql
--- USE beer

SELECT annee, numero_ticket, date_vente 
FROM ticket
WHERE MONTH(date_vente) = 3 AND YEAR(date_vente) = 2014;

```

### CODE 3

```sql
--- USE beer

SELECT annee, numero_ticket, date_vente 
WHERE date_vente LIKE "2014-03-%"

```
